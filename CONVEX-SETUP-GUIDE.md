# Convex Setup Guide for Mission Control

## Step 1: Create Project on Convex Dashboard

1. Go to https://dashboard.convex.dev
2. Click "Create a project"
3. Name it: **prep-mission-control**
4. Click "Create"

## Step 2: Get Your Deployment URL

Once created, you'll see:
- **Deployment URL** (looks like: https://something.convex.cloud)
- Copy this URL - I'll need it

## Step 3: Get Your Deploy Key

1. In the Convex dashboard, click "Settings"
2. Go to "Deploy Keys" 
3. Click "Generate Deploy Key"
4. Copy the key (starts with `prod:...`)

## Step 4: Paste Into Discord

Send me:
```
Deployment URL: [paste here]
Deploy Key: [paste here]
```

Then I'll:
- Set up the database schema
- Initialize Mission Control
- Deploy the 3-agent squad
- Give you the dashboard URL

---

**What you'll get:**
- Real-time Mission Control dashboard
- See all agent activity
- Track tasks in progress
- View completed work
- Monitor costs

---

**This takes ~2 minutes total. Then we're operational.**
