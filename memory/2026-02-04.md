## 💡 Key Decisions Made

### Architecture
- **SaaS over Services** - Automated, self-service, scalable
- **Railway for backend** - Auto-deploy from GitHub, easy env vars
- **Vercel for frontend** - Next.js optimized, global CDN
- **Neon for database** - Serverless PostgreSQL, free tier
- **Stripe LIVE mode** - Real payments from day 1

### Pricing Strategy (Official Ally Feb 2026)
- **Free $0** - Try Ally, 50 msg/day, 5 skills
- **Personal $9** - Entry point, low friction
- **Plus $19** - Power users, Gmail/Calendar
- **Pro $39** - Professionals, API access
- **Family $19** - 5 members, individual memories
- 14-day trial = lower resistance, higher conversion

### Tech Choices
- Next.js 14 (App Router) - Modern, fast, SEO-friendly
- Tailwind CSS - Rapid UI development
- Stripe - Industry standard, reliable
- Email templates - HTML for rich formatting
