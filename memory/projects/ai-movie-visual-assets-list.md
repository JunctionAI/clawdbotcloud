## Production Notes

### Consistency Requirements
- **Color Palette:** Define 3-5 core colors from film, use across all assets
- **Typography:** Lock in title font and secondary font, use consistently
- **Creator Likenesses:** Use same reference photos for consistency
- **Mood/Tone:** Maintain consistent genre feel (action, thriller, drama, etc.)

### Asset Specifications
| Platform | Dimensions | Format |
|----------|-----------|--------|
| Theatrical Poster | 27x40" (2700x4000px) | JPEG/PNG |
| Instagram Feed | 1080x1080px | JPEG |
| Instagram Story/Reel | 1080x1920px | JPEG/MP4 |
| Twitter Header | 1500x500px | JPEG |
| YouTube Thumbnail | 1280x720px | JPEG |
| YouTube Banner | 2560x1440px | JPEG |

### Generation Tips
1. **Batch similar assets** - Generate all character posters in one session for consistency
2. **Use consistent seed** - When possible, lock random seeds for visual coherence
3. **Reference real examples** - Study actual studio marketing campaigns for authenticity
4. **Build a style guide** - After first few assets, document what's working
