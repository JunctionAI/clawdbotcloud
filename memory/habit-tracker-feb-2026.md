# Habit Tracker - February 2026

**Purpose:** Track dopamine-driven habits (porn, weed, alcohol, coffee) to understand patterns and build a reduction plan over time.

**Approach:** 
- No forced resistance initially
- Just log the data honestly
- Look for patterns (triggers, stress, time of day)
- Build a gradual reduction plan based on insights

**Format:**
```
## YYYY-MM-DD

**Porn:** [Y/N] - Time: [HH:MM] - Trigger: [what were you feeling/avoiding?]
**Weed:** [Y/N] - Time: [HH:MM] - Trigger: [stress? flatmate? boredom?]
**Alcohol:** [Y/N] - Time: [HH:MM] - Trigger: [social? stress? routine?]
**Coffee:** [Y/N] - Time: [HH:MM] - Trigger: [energy? habit? social?]

**Pre-habit feeling:** [tense/bored/stressed/avoiding work/etc]
**Post-habit feeling:** [relaxed/guilty/sharper/duller/etc]
**Did it serve your goals?** [Y/N]
**Would $1M version of you do this?** [Y/N]
```
