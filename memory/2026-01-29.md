## CRITICAL LESSON: Email Formatting Failure ⚠️

**What happened:**
- Sent email to Andi Garnett (TWG client) with terrible formatting
- Email appeared as one giant wall of text with no paragraph breaks
- Tom had to immediately follow up with properly formatted version
- **This was embarrassing for Tom as an email marketing agency**

**The problem:**
I used Microsoft Graph API with plain text body and single line breaks (`\n`). Outlook didn't render these as paragraph breaks, creating an unreadable mess.

**What I sent (WRONG):**
```javascript
body: {
  contentType: "Text",
  content: "Hi Andi,\n\nGreat to connect!..."  // Single \n breaks
}
```

**What I SHOULD do going forward:**

### Option 1: HTML Email (PREFERRED for client emails)
```javascript
body: {
  contentType: "HTML",
  content: `<p>Hi Andi,</p>
<p>Great to connect! Happy to outline the process and get moving on this.</p>
<p>To streamline the migration from Mailchimp, here's what we'll need:</p>
<ul>
<li>Mailchimp admin access for me (tom@junctionmedia.ai or a TWG email if you prefer transparency)</li>
<li>Jon to create a Klaviyo account and add me as admin (same email address)</li>
</ul>
<p>Once I have access, I'll handle the migration...</p>`
}
```

### Option 2: Plain Text with Proper Breaks
```javascript
body: {
  contentType: "Text",
  content: `Hi Andi,\r\n\r\nGreat to connect!\r\n\r\nTo streamline...`
  // Use \r\n\r\n (CRLF + double break) for paragraph separation
}
```

**RULES FOR CLIENT EMAILS GOING FORWARD:**

1. **ALWAYS use HTML formatting** (`contentType: "HTML"`) for ALL emails
2. **Match Discord formatting exactly** - if it looks good in Discord, it must look identical in the email provider
3. **Use proper paragraph tags** (`<p>`) for structure
4. **Use `<br>` for line breaks** within paragraphs (like signature blocks)
5. **Use lists** (`<ul>`, `<li>`) when listing items
6. **Test first** - send to tom@junctionmedia.ai before sending to clients when unsure
7. **Remember:** Tom's reputation as an email marketing agency depends on professional formatting
