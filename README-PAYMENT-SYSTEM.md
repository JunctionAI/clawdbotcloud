## 🆘 Troubleshooting

### Payment link doesn't work
- Check email verification (NOWPayments/Gumroad)
- Verify link copied correctly
- Try incognito/different browser

### Customer confused about crypto
- NOWPayments has built-in instructions
- Shows QR code + wallet address
- Supports 200+ cryptocurrencies
- Consider Gumroad for credit cards (if added)

### Not receiving notifications
- Check spam folder
- Verify email in settings
- Test with small transaction
- Enable webhooks (advanced)
