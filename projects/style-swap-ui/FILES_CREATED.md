## 🚀 How to Use

### For Development
```bash
# 1. All files are already in the project
cd ~/clawd/projects/style-swap-ui

# 2. To use the enhanced app
# Edit src/main.jsx to import from AppEnhanced.jsx
# OR rename files:
mv src/App.jsx src/AppOriginal.jsx
mv src/AppEnhanced.jsx src/App.jsx

# 3. Install dependencies (if not already done)
npm install

# 4. Start dev server
npm run dev
```
