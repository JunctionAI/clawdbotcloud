# Social Media Post #7: User-Generated Content Feature (Instagram/TikTok)

## Platform
Instagram Feed/Reels + TikTok

## Format
1080x1920px (Reels) or 1080x1350px (Feed)

## Concept
Feature real user transformation with proper credit and engagement

---

## Visual Layout (Feed Post)

### Main Image
- User's before/after split screen
- High-quality, well-lit photo
- Clear transformation visible

### Overlay Elements
- "Community Spotlight" badge (top-left)
- User handle tag (bottom-left): "@[username]"
- Style Swap watermark (bottom-right, subtle)

### Text Overlay
- Top: "TRANSFORMATION OF THE WEEK"
- Middle: Brief quote from user
- Bottom: "Featured on @styleswap"

---

## Caption Template

```
[Repost/Regram 📸]

OBSESSED with @[username]'s transformation 😍

[She/He/They] tried on [X number] outfits and found the perfect [dress/jacket/outfit/style] for [occasion/event/everyday].

Swipe to see the before/after! ⬅️

💜 [Quote from user about their experience]

Want to be featured? 
1. Use #StyleSwap 
2. Tag @styleswap
3. Share your transformation!

Best submissions get featured weekly + win prizes 🎁

[Optional: Add specific detail about what made this transformation special - perfect fit, unexpected style, confidence boost, etc.]

#StyleSwap #CommunityLove #UserSpotlight #FashionTransformation #RealResults #BeforeAndAfter #StyleTransformation #FashionCommunity #UserFeature
```

---

## User Selection Criteria

### What Makes a Great UGC Feature

**Visual Quality:**
- Good lighting (natural or professional)
- Clear before/after distinction
- High resolution (not pixelated)
- Aesthetically pleasing composition

**Story Quality:**
- Compelling transformation (visual change)
- Authentic testimonial (genuine emotion)
- Relatable situation (others can see themselves)
- Specific details (numbers, occasions, results)

**Engagement Potential:**
- User has engaged audience (micro-influencer ideal)
- Unique angle or story (stands out)
- Inspirational or aspirational (motivates others)
- Diverse representation (body types, styles, ages)

## Weekly UGC Feature Process

### Monday: Hunt for Content
- Search #StyleSwap hashtag
- Review tagged posts (@styleswap mentions)
- Check DMs for submission requests
- Browse recent app reviews with photos

### Tuesday: Shortlist & Permission
- Select top 5-10 candidates
- DM users requesting permission to repost
- Template: "Hi [name]! We LOVE your Style Swap transformation. Can we feature you on our account this week? You'll be tagged and credited. Reply YES if interested! 💜"

### Wednesday: Create Feature Post
- Download user's content
- Design overlay graphics
- Write personalized caption
- Prepare Story content

### Thursday: Post & Promote
- Post to Feed/Reels
- Share to Stories with poll/question
- Tag user in post and Story
- Pin user's comment

### Friday: Engagement & Follow-Up
- Monitor comments and engagement
- Respond to all comments
- Feature user in weekly roundup
- Send thank you DM with promo code

## Legal & Permissions

### Permission Request Template (DM)
```
Hi [Name]! 👋

We absolutely LOVE your Style Swap transformation! Your [outfit/style/look] is stunning 😍

We'd love to feature you on our main account this week (2M+ followers). 

This includes:
✅ Feed/Reel post with full credit
✅ Story features
✅ Possible newsletter mention
✅ Entry into monthly giveaway ($500 prize)

Are you comfortable with us reposting? Just reply YES and we'll feature you [day]!

Also, any specific details you'd like us to include in the caption?

Thank you!
- The Style Swap Team 💜
```

### Rights Confirmation
- Screenshot permission approval
- Document in spreadsheet (date, user, post, permission granted)
- Store for 1 year minimum

## Variations

### Version A: Video UGC
If user shared a video try-on:
- Create 15-second highlight reel
- Add music and captions
- Feature as Reel with user's permission

### Version B: Multi-Outfit Feature
If user tried multiple looks:
- Create carousel showing 3-5 different try-ons
- Caption: "When you can't decide... try them ALL!"

### Version C: Before/After Journey
If user has transformation over time:
- Show progression (3 months, 6 months)
- Highlight style evolution
- Inspirational angle
