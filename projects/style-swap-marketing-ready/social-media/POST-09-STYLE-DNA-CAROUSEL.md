### From Post to Quiz
1. User views carousel
2. Identifies with a Style DNA
3. Clicks link in bio
4. Takes Style DNA quiz (10 questions)
5. Gets personalized result + recommendations
6. Prompted to try on recommended outfits
7. Downloads app to see results

### Quiz Landing Page Elements
- Engaging visual quiz (interactive)

- Instant results with detailed breakdown
- Personalized outfit recommendations
- CTA: "See these outfits on YOU" (app download)
