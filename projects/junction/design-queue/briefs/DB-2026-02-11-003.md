## 🎯 Project Overview

### What is this for?
Reusable social post templates for Tom's LinkedIn thought leadership content about AI marketing.

### Context
Tom will be posting regularly on LinkedIn to build authority and drive leads to Junction Media. We need a set of templates that can be quickly customized for different content types while maintaining consistent brand recognition.

These templates will be used by AI to generate post graphics, so they need to be simple enough for quick iteration but distinctive enough to stand out in the feed.

### Where will it be used?
- [ ] Website
- [x] Social Media (LinkedIn primary, adaptable to Twitter/X)
- [ ] Email
- [ ] Presentation
- [ ] Print
- [ ] Video thumbnail

## 🎨 Creative Direction

### Visual Style
- **Bold typography** — Text-forward, minimal imagery
- **Signature gradient element** — Consistent across all templates
- **Clean negative space** — Easy to read quickly
- **Professional but not boring** — Stand out in LinkedIn's sea of blue

### Template System
Each template should have:
1. **Fixed elements:** Logo position, gradient accent, overall layout
2. **Variable elements:** Text content, optional supporting graphic
3. **Color variants:** Dark background + Light background options

### Signature Brand Element
Create a distinctive visual element that appears on all Junction social posts:
- Could be: gradient line, corner gradient, subtle mesh pattern
- Should be: immediately recognizable as Junction
- Must be: non-intrusive to content

### Mood/Tone
- **Confident** — We know what we're talking about
- **Forward-thinking** — Cutting edge, not catching up
- **Premium** — High-end agency feel, not startup hustle

### Reference Inspiration
1. Chris Do (The Futur) — Bold typographic social posts
2. Alex Hormozi — High-contrast, text-forward
3. Sahil Bloom — Clean, readable, professional

### What to Avoid
- ❌ Generic LinkedIn blue aesthetic
- ❌ Overly complex graphics
- ❌ Hard-to-read fonts or small text
- ❌ Cheesy stock imagery
- ❌ Too many colors or elements
