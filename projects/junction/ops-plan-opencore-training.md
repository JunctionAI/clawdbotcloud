## Service 1: OpenCore Setup

### Overview
Deploy custom AI agent infrastructure for client businesses — autonomous systems that handle repetitive tasks, data processing, and operational workflows.

### Target Client Profile
- **Business size:** 10-200 employees
- **Pain points:** Manual processes, scaling bottlenecks, staff doing repetitive work
- **Tech readiness:** Basic (we handle the technical complexity)
- **Budget:** $5k-$20k setup + ongoing support

### Service Tiers

#### Tier 1: Starter ($5,000 setup)
- **1-2 AI agents** configured
- **Use cases:** Email triage, data entry automation, basic reporting
- **Includes:**
  - Initial discovery call (2 hours)
  - System architecture design
  - Agent deployment & testing
  - 2 weeks of tuning
  - Documentation & handoff
  - 30 days support

#### Tier 2: Professional ($10,000 setup)
- **3-5 AI agents** with interconnections
- **Use cases:** Customer service automation, lead qualification, content scheduling, inventory alerts
- **Includes:**
  - Full workflow audit (half-day)
  - Custom agent development
  - Integration with existing tools (CRM, email, Slack, etc.)
  - 4 weeks of tuning
  - Staff training session (2 hours)
  - 60 days support

#### Tier 3: Enterprise ($20,000+ setup)
- **6+ AI agents** forming an autonomous system
- **Use cases:** End-to-end process automation, multi-department workflows, decision support systems
- **Includes:**
  - Comprehensive operations audit (1-2 days)
  - Custom agent ecosystem
  - Full integration suite
  - Ongoing optimization (8 weeks)
  - Team training program
  - Dedicated support channel
  - 90 days support + priority access

### Delivery Process

#### Phase 1: Discovery (Week 1)
1. **Kickoff call** — Understand business, pain points, goals
2. **Workflow mapping** — Document current processes
3. **Opportunity identification** — Where AI agents add most value
4. **Scope definition** — Agree on agents to build
5. **Deliverable:** Discovery Report + Project Plan

#### Phase 2: Design (Week 2)
1. **Architecture design** — Agent structure, data flows, integrations
2. **Tool selection** — Platforms, APIs, infrastructure
3. **Security review** — Data handling, access controls
4. **Client approval** — Sign off on design
5. **Deliverable:** Technical Specification Document

#### Phase 3: Build (Weeks 3-4)
1. **Environment setup** — Infrastructure, accounts, connections
2. **Agent development** — Build & configure each agent
3. **Integration work** — Connect to client systems
4. **Internal testing** — QA before client exposure
5. **Deliverable:** Deployed agents (staging)

#### Phase 4: Test & Tune (Weeks 5-6)
1. **Client UAT** — User acceptance testing
2. **Edge case handling** — Refine based on real scenarios
3. **Performance optimization** — Speed, accuracy, reliability
4. **Staff shadowing** — Agents work alongside humans
5. **Deliverable:** Production-ready system

#### Phase 5: Launch & Handoff (Week 7)
1. **Go-live** — Agents active in production
2. **Monitoring setup** — Alerts, dashboards, logging
3. **Documentation** — How everything works
4. **Staff training** — Using and managing the system
5. **Deliverable:** Complete handoff package

#### Phase 6: Support (Ongoing)
1. **Active monitoring** — First 30-90 days
2. **Bug fixes** — Rapid response to issues
3. **Optimization** — Continuous improvement
4. **Check-ins** — Weekly during support period
5. **Transition** — To maintenance mode or ongoing partnership

### Required Resources

**Per Project:**
- **Lead Architect:** 20-40 hours (scoping, design, oversight)
- **AI Developer:** 40-80 hours (build, test, deploy)
- **Integration Specialist:** 10-20 hours (API connections)
- **Project Manager:** 10-15 hours (coordination)

**Tools & Platforms:**
- Clawdbot / OpenAI / Anthropic APIs
- Make.com / Zapier for integrations
- Cloud hosting (client pays directly or included)
- Communication: Slack/Discord channel per project

| Starter | $5,000 | $500/mo | 70% |

## Service 2: AI Training for Staff

### Overview
Hands-on training programs to upskill client teams on AI tools, prompt engineering, and workflow automation. Transform "AI curious" employees into "AI proficient" operators.

### Target Client Profile
- **Business size:** Any (5-500+ employees)
- **Pain points:** Staff not using AI, productivity gaps, fear of AI, inconsistent adoption
- **Existing tools:** Often have ChatGPT/Copilot but underutilizing
- **Budget:** $2k-$15k depending on scope

### Training Packages

#### Package A: AI Essentials Workshop ($2,000)
**Duration:** Half-day (4 hours)
**Participants:** Up to 20 people
**Format:** In-person or virtual

**Curriculum:**
1. **AI Landscape** (30 min) — What's possible today, what's hype
2. **Hands-On: ChatGPT/Claude** (90 min) — Effective prompting, real exercises
3. **Use Case Mapping** (60 min) — Finding AI opportunities in their roles
4. **Ethics & Safety** (30 min) — What not to do, data privacy
5. **Q&A & Action Planning** (30 min)

**Deliverables:**
- Slide deck (customized for their industry)
- Prompt template library
- Resource guide
- Recording (if virtual)

#### Package B: Team Transformation Program ($7,500)
**Duration:** 3 sessions over 3 weeks
**Participants:** Up to 15 people (deeper engagement)
**Format:** In-person or virtual

**Week 1: Foundations (4 hours)**
- AI capabilities & limitations
- Prompt engineering masterclass
- Tool comparison (ChatGPT vs Claude vs Gemini vs Copilot)
- Homework: Apply to one task

**Week 2: Advanced Applications (4 hours)**
- Multi-step prompting & chains
- Document analysis & synthesis
- Image generation basics
- Custom GPTs / Claude Projects
- Homework: Build a workflow

**Week 3: Integration & Automation (4 hours)**
- Connecting AI to existing tools
- Basic automation with Make/Zapier
- Team workflow design
- Measuring ROI
- Certification & next steps

**Deliverables:**
- Full curriculum materials
- Video recordings
- Prompt library (50+ templates)
- Automation templates
- Certificates of completion
- 30 days email support

#### Package C: Enterprise Training Program ($15,000+)
**Duration:** Custom (typically 6-8 weeks)
**Participants:** Multiple cohorts, up to 100+ people
**Format:** Hybrid (in-person kickoff + virtual sessions)

**Includes:**
- Needs assessment & skill gap analysis
- Custom curriculum development
- Department-specific tracks (Sales, Marketing, Ops, HR, etc.)
- Train-the-trainer component
- Internal champion development
- Ongoing office hours
- Progress tracking & reporting
- Executive briefings

**Deliverables:**
- Complete training program (reusable)
- Internal knowledge base
- Champion certification
- Quarterly refresher sessions
- ROI measurement framework

### Delivery Process

#### Pre-Training
1. **Discovery call** — Understand team, current AI usage, goals
2. **Skill assessment** — Survey to gauge baseline
3. **Content customization** — Tailor examples to their industry
4. **Logistics** — Schedule, platform setup, materials

#### During Training
1. **Interactive delivery** — Not lectures, hands-on exercises
2. **Real-world examples** — Use their actual tasks
3. **Breakout sessions** — Small group practice
4. **Live Q&A** — Address specific challenges
5. **Capture insights** — Note common struggles for follow-up

#### Post-Training
1. **Materials delivery** — All resources within 48 hours
2. **Feedback collection** — Survey participants
3. **Follow-up session** — Optional 1-hour check-in (2 weeks later)
4. **Support period** — Email questions for 30 days
5. **Impact measurement** — Productivity metrics where possible

### Required Resources

**Per Workshop:**
- **Lead Trainer:** 8-12 hours (prep + delivery + follow-up)
- **Materials Designer:** 4-6 hours (customization)
- **Tech Support:** 2 hours (for virtual sessions)

**Tools:**
- Presentation software (Pitch/Canva)
- Virtual platform (Zoom/Teams)
- Practice environment (sandbox accounts)
- Assessment tools (Typeform)
