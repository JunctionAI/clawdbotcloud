# Graphics Folder

Place your generated marketing graphics here.

## Expected Files

Once you generate the graphics using the specifications in `../MARKETING-GRAPHICS-SPECS.md`, place them here with these filenames:

```
graphics/
├── instagram-post-1080x1080.png
├── instagram-story-1080x1920.png
├── twitter-post-1200x675.png
├── facebook-ad-1200x628.png
├── linkedin-banner-1200x627.png
├── tiktok-thumbnail-1080x1920.png
├── pinterest-pin-1000x1500.png
└── youtube-thumbnail-1280x720.png
```

## Quick Generate

### Using ChatGPT Plus / DALL-E 3
1. Open ChatGPT Plus
2. Copy prompts from `MARKETING-GRAPHICS-SPECS.md`
3. Paste one at a time
4. Download each image
5. Rename to match filenames above
6. Place in this folder

### Using Midjourney
1. Join Midjourney Discord
2. Adapt prompts for Midjourney syntax
3. Use `/imagine` command
4. Download generated images
5. Place in this folder

### Using Canva
1. Go to Canva.com
2. Create custom size (use dimensions from specs)
3. Design following specifications
4. Export as PNG
5. Place in this folder

---

**Tip**: Keep source files (PSD, Figma, Canva links) for future edits!
