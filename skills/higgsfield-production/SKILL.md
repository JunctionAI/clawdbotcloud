# Higgsfield Autonomous Production Skill

## Overview
Produce image-to-video content using Higgsfield without file system access. All assets stay in "recently generated" within the browser session.

## Platform URLs
- Main: https://higgsfield.ai/
- Character generation: Character menu → Select character → Generate
- Video creation: Video menu → Create Video

## Workflow: Character-Based Scene Production

### Step 1: Generate Scene Image
1. Navigate to **Character** (top nav)
2. Find target character (e.g., "InfluencerTom")
3. Click **Generate** button on that character
4. Opens Higgsfield Soul with character pre-selected
5. Enter scene prompt in textbox
6. Set aspect ratio (16:9 for cinematic, 9:16 for vertical)
7. Set quality (2k recommended)
8. Click **Generate** (2 credits per image)
9. Image appears in gallery above

### Step 2: Animate Image to Video
1. Click on the generated image in gallery
2. Dialog opens with options
3. Click **Animate** button
4. Opens Video creation with image as start frame
5. Enter motion/action prompt
6. Select model (Kling 2.6 = 10 credits with audio, Kling 2.5 Turbo = 6 credits)
7. Set duration (5s standard)
8. Click **Generate**
9. Video appears in History tab

### Step 3: Repeat for All Scenes
- Each scene = 1 image + 1 video
- Images stay in recently generated
- Videos stay in History
- No file access needed

## Cost Estimates
- Image generation: 2 credits (Higgsfield Soul)
- Video generation: 6-10 credits (depending on model/duration)
- Per scene total: ~8-12 credits
- 7-scene film: ~60-84 credits

## Prompting Best Practices

### Image Prompts
- Be specific about: pose, expression, clothing, environment, lighting
- Include: "cinematic lighting", "shallow depth of field", "golden hour"
- Example: "InfluencerTom sitting at a laptop in a modern café in Lisbon, focused expression, morning light through windows, shallow depth of field, cinematic"

### Video Motion Prompts
- Describe the ACTION, not just the scene
- Include: camera movement, character actions, atmospheric changes
- Example: "Camera slowly pushes in. Tom looks up from laptop with a satisfied smile. Sunlight shifts. Background patrons move naturally. Ambient café sounds."

## Autonomous Production Checklist

Before starting overnight production:
- [ ] Verify credits available (check bottom right corner)
- [ ] Note character names to use
- [ ] Prepare all scene prompts in advance
- [ ] Prepare all motion prompts in advance
- [ ] Test one scene end-to-end first
- [ ] Clear any stuck generations from queue

## Troubleshooting

### "NSFW detected" / "Restricted content"
- Adjust prompt to be less suggestive
- Try different model (Wan is more lenient)
- Credits are refunded for blocked content

### Failed generation
- Credits are refunded
- Try different prompt or model
- Check if character has enough training images

### Images not appearing
- Refresh page
- Check if generation is still processing
- Look in Asset Library for stuck assets
