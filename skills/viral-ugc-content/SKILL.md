# Viral UGC Content Skill

Create short-form content engineered for virality using proven patterns.

## The 3-Second Rule

**You have 3 seconds to stop the scroll.** Everything else is secondary.

### Hook Types That Work

1. **Pattern Interrupt** — Unexpected visual/audio in first frame
2. **Open Loop** — Create curiosity that MUST be resolved
3. **Controversial Take** — Disagree with common belief
4. **POV Format** — "POV: [relatable/interesting situation]"
5. **Direct Challenge** — "Stop scrolling if you [specific trait]"
6. **Numbers/Lists** — "3 things nobody tells you about..."
7. **Before/After** — Transformation promise

## Viral Content DNA

### 1. NATIVE FEEL (Critical)
- Shot on phone, not cinema camera
- Imperfect = authentic
- Real person talking, not voiceover
- Raw energy > polish

### 2. TEXT ON SCREEN
- Smart captions that add humor/context
- Key words highlighted
- Text reveals timed to speech
- Meme-style commentary

### 3. FACE ON CAMERA
- Human faces = 38% higher engagement
- Eye contact with lens
- Expressive reactions
- Real emotion > acting

### 4. SOUND STRATEGY
- Trending sounds boost discovery
- Original audio can go viral
- Voice + music layered
- Satisfying audio moments

### 5. PACING
- Fast cuts (2-4 seconds per shot)
- No dead air
- Build to payoff
- End before overstaying

## Content Formats That Go Viral

### Talking Head + B-Roll
```
[Face to camera hook] → [B-roll showing thing] → [Back to face for payoff]
```

### POV Format
```
Text: "POV: [situation]"
Visual: First-person experience of that situation
```

### Story Time
```
Hook → Setup → Tension → Twist/Payoff → CTA
```

### React/Duet
```
Original content on one side, reaction on other
Real-time genuine response
```

### Before/After Split
```
Left: Before state
Right: After transformation
Swipe or reveal moment
```

### The Reveal
```
Build anticipation → Dramatic pause → Show the thing
```

## Hook Templates

### Curiosity Hooks
- "Nobody's talking about this..."
- "I found something insane..."
- "Wait for it..."
- "This changes everything..."

### Challenge Hooks
- "Stop scrolling if you..."
- "Only [X]% of people know this..."
- "You've been doing this wrong..."

### Story Hooks
- "So this just happened..."
- "I can't believe I'm sharing this..."
- "The craziest thing about [X]..."

### POV Hooks
- "POV: you just discovered..."
- "POV: it's 2am and..."
- "POV: you're the main character..."

## Engagement Triggers

1. **Curiosity Gap** — Create questions that need answers
2. **Relatability** — "This is SO me"
3. **Controversy** — Polarizing takes get comments
4. **FOMO** — Limited time/exclusive info
5. **Aspiration** — "I want that life"
6. **Nostalgia** — Throwback emotions
7. **Shock/Surprise** — Unexpected moments
8. **Humor** — Funny > everything

## Platform Optimization

### TikTok
- 15-60 seconds optimal
- Use trending sounds
- Post 1-3x daily
- Reply to comments with videos

### Instagram Reels
- 15-30 seconds optimal
- Strong thumbnail
- Hashtag strategy (5-10)
- Share to Stories

### YouTube Shorts
- Up to 60 seconds
- Strong hook AND ending
- Subscribe CTA works
- Searchable titles

## What NOT To Do

❌ Over-produced content (looks like an ad)
❌ Slow intros (lose viewers instantly)
❌ No text on screen (accessibility + scanning)
❌ Generic CTA ("like and follow")
❌ Ignoring trends (missing discovery)
❌ Perfect = boring (embrace imperfection)
