# Viral Hook Formulas

## Pattern 1: The Open Loop
Creates irresistible curiosity.

```
"I [did thing] and you won't believe what happened..."
"There's something about [topic] nobody's talking about..."
"I figured out why [common problem] keeps happening..."
```

## Pattern 2: The Controversial Take
Forces engagement (agree or fight).

```
"Unpopular opinion: [controversial statement]"
"[Common advice] is actually terrible advice..."
"Everyone's wrong about [topic]..."
```

## Pattern 3: The Direct Address
Targets specific viewer.

```
"If you're a [specific person], watch this..."
"Stop scrolling if you've ever [relatable thing]..."
"This is for everyone who [specific situation]..."
```

## Pattern 4: The Transformation Promise
Shows value immediately.

```
"How I went from [bad state] to [good state] in [time]..."
"Watch me turn [thing] into [better thing]..."
"Before and after [doing thing]..."
```

## Pattern 5: The Story Tease
Human brain can't resist stories.

```
"So I was [doing normal thing] when [unusual thing happened]..."
"The craziest thing happened to me [timeframe]..."
"Let me tell you about the time I [interesting situation]..."
```

## Pattern 6: The List/Number
Promises structured value.

```
"3 things I wish I knew about [topic]..."
"The 5 biggest mistakes people make with [thing]..."
"Here are [X] signs that [situation]..."
```

## Pattern 7: The POV
Immersive experience.

```
"POV: you just found out [discovery]..."
"POV: you're [interesting situation]..."
"POV: it's [time] and you're [doing thing]..."
```

## Pattern 8: The Secret/Hack
Everyone wants insider knowledge.

```
"The [industry] secret they don't want you to know..."
"I found a hack for [common problem]..."
"Here's what [experts] won't tell you..."
```

## Pattern 9: The Question
Forces mental engagement.

```
"Have you ever wondered why [thing]?"
"What would you do if [scenario]?"
"Am I the only one who [relatable thing]?"
```

## Pattern 10: The Warning
Triggers loss aversion.

```
"Stop doing [common thing] immediately..."
"If you're [doing thing], you need to see this..."
"The mistake that's costing you [valuable thing]..."
```

## ALTERNATE-Specific Hooks

### AI Film Angle
```
"We made a movie with AI in 7 days for $50..."
"POV: you find an app that shows your alternate lives..."
"What if AI could show you the life you didn't choose?"
"I asked AI to show me my other lives. This is what happened..."
```

### Existential/Philosophical Angle
```
"Would you want to see the life you almost had?"
"The question that kept me up all night..."
"What if the life you're living isn't the one you'd choose?"
```

### Behind-the-Scenes Angle
```
"How we made a movie without cameras, actors, or a budget..."
"Day 1 vs Day 7 of making an AI film..."
"The tools we used to make a full movie for $50..."
```

### Meta/Self-Aware Angle
```
"We used AI to make a movie about AI showing alternate lives..."
"Is this the weirdest movie ever made? Probably..."
"When the AI started generating scenes we didn't expect..."
```
