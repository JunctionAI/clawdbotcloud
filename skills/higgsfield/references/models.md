# Higgsfield Models Reference

## Video Generation Models

### Text-to-Video
- **Input:** Text prompt only
- **Output:** 4-8 second video clip
- **Best for:** Scene establishing, abstract visuals, motion graphics
- **Gen time:** 2-5 minutes

### Image-to-Video
- **Input:** Static image + motion prompt
- **Output:** 4-8 second animated video
- **Best for:** Bringing still images to life, consistent character animation
- **Gen time:** 2-4 minutes

### Video-to-Video
- **Input:** Existing video + style/modification prompt
- **Output:** Transformed video
- **Best for:** Style transfer, visual effects, enhancement
- **Gen time:** 3-6 minutes

## Image Generation

### Flux-based Image Gen
- **Input:** Text prompt
- **Output:** High-quality still image
- **Best for:** Concept art, scene stills, character designs
- **Gen time:** 30-90 seconds

## Recommended Settings

### Cinematic Content
- Model: Text-to-Video or Image-to-Video
- Aspect: 16:9
- Style: Cinematic, film grain
- Motion: Slow-medium (prevents artifacts)

### Social Media Content
- Model: Image-to-Video (for consistent branding)
- Aspect: 9:16 (Stories/Reels) or 1:1 (Feed)
- Style: Bold, high contrast
- Motion: Dynamic

### Artistic/Abstract
- Model: Text-to-Video
- Aspect: Variable
- Style: Experimental prompts work well
- Motion: Can go wild
