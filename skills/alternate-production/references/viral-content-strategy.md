# ALTERNATE Viral Content Strategy

## The Truth About Promo Content

**What doesn't work:**
- Pretty cinematic clips with no context
- Generic "coming soon" posts
- Over-produced trailer cuts

**What DOES work:**
- Real humans talking about real things
- Smart text overlays with humor
- Native UGC feel
- Meta/self-aware content
- Behind-the-scenes authenticity

---

## Content Pillars for ALTERNATE

### 1. THE META LAYER (Primary Angle)
The movie is self-aware. The content should be too.

**Concepts:**
- "We used AI to make a movie about AI" — the inception angle
- "The AI generated things we didn't ask for" — the mystery
- "Made for $50, no cameras, no actors" — the disruption

**Format:** Talking head + movie clips intercut

### 2. THE EXISTENTIAL QUESTION
The movie asks big questions. So should the content.

**Concepts:**
- "Would you want to see your alternate life?"
- "What if you're living the wrong timeline?"
- "The choice that changes everything"

**Format:** Text on screen, atmospheric clips, trending sound

### 3. BEHIND THE SCENES
People love seeing how things are made, especially with AI.

**Concepts:**
- "Day 1 vs Day 7" transformation
- "The tools that made this possible"
- "When the AI surprised us"
- "Prompt → Result reveals"

**Format:** Screen recordings, before/after, reaction clips

### 4. CHARACTER MOMENTS
Humanize the AI-generated characters.

**Concepts:**
- Character introduction posts
- "Which alternate life would you choose?"
- Scene breakdowns
- POV content from character perspective

**Format:** Clips with smart captions, polls, engagement bait

## Content Types to Create

### TYPE A: Talking Head + Clips
```
[Tom/Zac to camera] "We made a movie with AI..."
[Cut to epic shot from film]
[Back to face] "...and it only cost $50"
[More clips]
[Face] "Link in bio"
```
**Length:** 30-60 seconds
**Vibe:** Authentic, slightly disbelief, excited

### TYPE B: POV Experience
```
Text: "POV: you find an app that shows your alternate lives"
[Clip: friends looking at phone]
[Clip: portal/mandala effect]
[Clip: cyberpunk scene]
[Clip: back to friends, shocked faces]
Text: "GENERATE AGAIN?"
```
**Length:** 15-30 seconds
**Vibe:** Immersive, mysterious

### TYPE C: The Reveal
```
[Black screen] "We made a movie for $50"
[Quick flash of scenes]
"No cameras"
[More scenes]
"No actors"
[More scenes]
"Just AI"
[Epic wide shot]
"ALTERNATE - Feb 14"
```
**Length:** 15-20 seconds
**Vibe:** Hype, build-up, payoff

### TYPE D: Smart Caption Style
```
[Clip plays]
Caption: "When the AI asks if you want to see another life"
[New clip]
Caption: "And you've already seen 47"
[Character looking exhausted]
Caption: "But you hit generate again anyway"
```
**Length:** 10-20 seconds
**Vibe:** Funny, relatable, meme-y

### TYPE E: Before/After or Split Screen
```
Left: "Day 1 - we have an idea"
Right: "Day 7 - we have a movie"
[Split screen comparison]
```
**Length:** 10-15 seconds
**Vibe:** Transformation, impressive

## Hooks to Use

### Curiosity Hooks
- "We made a movie with AI and something weird happened..."
- "I need to tell you about this film we created..."
- "What if AI could show you the life you didn't choose?"

### Direct Hooks  
- "If you've ever wondered 'what if' — watch this"
- "Stop scrolling if you believe in alternate realities"

### Meta Hooks
- "We used AI to make a movie about AI. Yes, it gets weird."
- "Made a film for $50. Here's everything that went wrong (and right)"

## Platform-Specific Notes

### TikTok
- Lead with talking head or text hook
- Use trending sounds when relevant
- Keep under 45 seconds for completion rate
- Encourage duets/stitches

### Instagram Reels
- Strong visual thumbnail
- Text on screen essential
- Can be slightly more polished
- Use carousel for behind-the-scenes

### YouTube Shorts
- Can be more "documentary" style
- Searchable titles matter
- End with clear CTA
- Subscribe ask works here
