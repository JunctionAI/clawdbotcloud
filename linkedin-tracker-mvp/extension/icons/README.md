# Extension Icons

Create 3 icon files in this directory:
- icon16.png (16x16px)
- icon48.png (48x48px)
- icon128.png (128x128px)

## Quick Icon Generation

You can use Canva, Figma, or any icon generator. 

**Design guidelines:**
- Color scheme: Blue/purple gradient (#667eea to #764ba2)
- Icon: Chart/graph symbol or LinkedIn-style icon
- Clean, professional look

**Free online tools:**
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Canva (free account)

**Or use this emoji-to-image approach:**
1. Go to https://www.favicon.cc/
2. Create a simple chart icon
3. Export in 3 sizes

For MVP, you can temporarily use simple colored squares, but replace before Chrome Web Store submission.
