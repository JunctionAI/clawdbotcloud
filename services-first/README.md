# Services-First Model - Complete Sales Kit

**Strategy:** Manual Clawdbot setup services, NOT SaaS product.

## What's in This Folder

### 📄 `services-sales-page.html`
Beautiful, conversion-focused landing page.
- Hero: "We Build Your AI Operations Team"
- Problem → Solution → Pricing → Case Study → CTA
- Ready to deploy (just add your Calendly link)
- Mobile responsive, modern design

### 📦 `service-packages.md`
Detailed breakdown of all service tiers.
- Starter: $1K setup + $500/mo
- Pro: $2K setup + $1K/mo (most popular)
- Enterprise: $5K setup + $2K/mo
- What's included, timelines, add-ons, FAQ

### ✉️ `outreach-templates.md`
Cold email, LinkedIn, and call scripts.
- 3 email templates (Founder/CEO, Salesperson, Operator)
- LinkedIn outreach sequence
- Cold call script
- Follow-up sequence (Day 3, 7, 14)
- Key messaging guidelines

### 📞 `discovery-call-script.md`
Complete call framework (15-30 min).
- Discovery questions to qualify and find pain
- Demo script using their specific workflow
- Objection handling
- Close techniques
- Follow-up email templates

## Quick Start

1. **Deploy Sales Page:**
   - Update Calendly links in `services-sales-page.html`
   - Host on your domain
   - Drive traffic via outreach

2. **Start Outreach:**
   - Customize templates in `outreach-templates.md`
   - Use your voice, your client results
   - CTA: Book discovery call

3. **Run Discovery Calls:**
   - Follow `discovery-call-script.md`
   - Qualify → Demo → Close
   - Send contract + invoice same day

4. **Deliver Service:**
   - Reference `service-packages.md` for scope
   - Setup in 1-4 weeks depending on tier
   - Monthly retainer for ongoing support
