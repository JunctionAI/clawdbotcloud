## Service Delivery Process

### Phase 1: Discovery Call (30 min, free)

**Goal:** Assess fit, understand workflows, recommend tier

**Questions to Ask:**
1. What's your biggest time-waster right now? (email? scheduling? research?)
2. How many hours per week do you spend on admin/busywork?
3. What tools do you use daily? (email, calendar, CRM, project management)
4. What would getting 10+ hours back per week unlock for you?
5. Have you tried AI tools before? What worked/didn't work?
6. What's your team size? (1 person vs 20 people = different tiers)
7. Any compliance or security requirements? (HIPAA, SOC2, etc.)

**Output:** Tier recommendation + custom proposal

**Booking:** Calendly link (30-minute slots)

---

### Phase 2: Proposal & Contract (24 hours after call)

**Deliverables:**
- Custom proposal (PDF)
  - Summary of their pain points
  - Recommended tier + justification
  - Custom scope (specific skills, integrations)
  - ROI calculation (specific to their hourly rate)
  - Timeline
  - Pricing breakdown
- Service agreement (DocuSign)
- Invoice for setup fee (due before work begins)

**Payment Terms:**
- Setup fee: 100% upfront (before work begins)
- Monthly retainer: Billed on 1st of each month (autopay via Stripe)

---

### Phase 3: Kickoff & Setup (48 hours - 2 weeks)

**Day 1: Kickoff Call (1 hour)**
- Deep dive on workflows
- Gather credentials (email, calendar, messaging apps)
- Confirm integrations needed
- Set expectations for timeline

**Day 2-3: Technical Setup (our work)**
- Install Clawdbot on infrastructure
- Configure email, calendar, messaging integrations
- Install skills (tier-specific)
- Setup memory system
- Configure security protocols
- Test all integrations
- Create custom documentation

**Day 4-5: Custom Configuration (our work)**
- Develop custom skills (Pro/Enterprise)
- Train AI on communication style (sample emails)
- Configure Heartbeat schedule (Pro/Enterprise)
- Setup Mission Control (Pro/Enterprise)
- Enterprise-specific work (SSO, audit logs, etc.)

**Day 6-7: Testing & QA**
- Internal testing of all features
- Send test messages, verify responses
- Check all integrations working
- Prepare training materials

---

### Phase 4: Training & Handoff (1-2 hours)

**Training Session 1 (1 hour):**
- Walkthrough of interface (WhatsApp/Telegram/Slack)
- Demonstrate core skills
- Practice common workflows together
- Answer questions
- Record session for reference

**Training Session 2 (Pro/Enterprise only, 1 hour):**
- Advanced features (Mission Control, Heartbeat)
- Custom skill demonstration
- Team training (if multi-user)
- Advanced workflows

**Handoff:**
- Credentials delivered
- Documentation shared (Google Doc)
- Support channels setup (email or Slack)
- Schedule first monthly check-in

---
