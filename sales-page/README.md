# Clawdbot Premium Sales Page

High-converting landing page for premium Clawdbot setups.

## 🚀 Deploy to Netlify (1 minute)

1. Drag and drop the `sales-page` folder to [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Done! Your page is live.

## 🔧 Required Customizations

Before deploying, update these placeholders:

### Payment Links
- **Line 408**: Replace `https://buy.stripe.com/your-payment-link` with your Stripe payment link

### Calendly Integration
- **Line 449**: Replace `https://calendly.com/your-calendly-link/30min` with your Calendly link

### GitHub/Docs Links
- **Line 190**: Update GitHub repo link
- **Line 476**: Update GitHub link in footer
- **Line 477**: Add documentation link

### Optional Customizations
- **Logo**: Replace the emoji (🤖) with a real logo image
- **Testimonials**: Add more real user testimonials as you get them
- **Screenshots**: Replace the terminal mockup with actual Mission Control screenshots
- **Domain**: Point a custom domain in Netlify settings
