## 🎯 What This Is

A **complete end-to-end automation system** for freelance/consulting services:

- **Lead capture** (landing pages + forms)
- **Automated proposals** (beautiful HTML in 60 seconds)
- **Pricing calculator** (value-based + hourly)
- **Client onboarding** (email sequences, payment tracking)
- **Testimonials & referrals** (automatic collection)
- **Upsell triggers** (based on client history)
- **Analytics dashboard** (track everything)

---

## 🚀 Quick Start

**Read this first:** `START_HERE.md`

**Then:**
1. `cd backend && npm install`
2. `npm run init-db`
3. `npm start`
4. Open `frontend/landing-clawdbot.html` in browser
5. Test form submission
6. Deploy (see `docs/DEPLOYMENT.md`)
