## Email Provider
**Recommended: Resend**
- Great DX
- React email templates
- Good deliverability
- Reasonable pricing

**Alternatives:**
- Postmark (excellent deliverability)
- SendGrid (enterprise scale)
- AWS SES (cheapest, more setup)

## Technical Approach

### Email Service Setup
```typescript
// Using Resend + React Email
import { Resend } from 'resend';
import { VerifyEmail } from '@/emails/verify-email';

const resend = new Resend(process.env.RESEND_API_KEY);

await resend.emails.send({
  from: 'Ally <hello@ally.ai>',
  to: user.email,
  subject: 'Verify your account',
  react: VerifyEmail({ token: verificationToken }),
});
```

### Template Organization
```
emails/
```

### Database Schema
```sql
email_logs:
  id: uuid PK
  user_id: uuid FK
  type: string (verify, reset, etc)
  to: string
  subject: string
  status: enum (sent, delivered, bounced, failed)
  sent_at: timestamp
  
email_preferences:
  user_id: uuid PK FK
  marketing: boolean default true
  product_updates: boolean default true
  weekly_digest: boolean default true
```

## Deliverability Checklist

- [ ] Dedicated sending domain

3. Templates render correctly across clients
4. Unsubscribe works instantly

6. All transactional emails have fallback text version
