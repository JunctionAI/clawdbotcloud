# Task: Brand identity and design system

## Description

Establish Ally's complete brand identity including logo, color palette, typography, voice/tone guidelines, and design system components. This foundational work enables all other visual and marketing tasks.

## Acceptance Criteria

- [ ] Logo designed (primary + icon versions)
- [ ] Color palette defined (primary, secondary, accents, semantic colors)
- [ ] Typography selected (headings, body, UI)
- [ ] Voice/tone guidelines documented
- [ ] Design system tokens exported (CSS variables, Figma tokens)
- [ ] Icon set selected or designed
- [ ] Component library foundations (buttons, inputs, cards)

## Technical Details

### Brand Elements
- **Logo:** Wordmark "Ally" + companion icon/symbol
- **Colors:** Warm, approachable palette (avoid cold tech blue)
- **Tagline:** "Life, handled." prominently featured
- **Tone:** Friendly, capable, trustworthy, never condescending

### Deliverables
- `/brand/logo/` - Logo files (SVG, PNG at various sizes)
- `/brand/colors.css` - CSS custom properties
- `/brand/typography.css` - Font imports and scales
- `/brand/GUIDELINES.md` - Brand usage documentation
- Figma file with design system components

### Implementation Notes
- Use CSS custom properties for theming
- Consider dark mode from the start
- Ensure WCAG 2.1 AA contrast compliance
- Design for mobile-first
