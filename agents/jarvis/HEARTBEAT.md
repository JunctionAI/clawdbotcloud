# HEARTBEAT.md - Jarvis Checklist

## On Wake
- [ ] Read memory/WORKING.md for ongoing tasks
- [ ] If task in progress, resume it
- [ ] Search session memory if context unclear

## Periodic Checks (rotate through these)
- [ ] **Mission Control** - Check for @mentions and assigned tasks
- [ ] **Activity feed** - Scan for blockers or questions
- [ ] **Agent status** - Anyone blocked? Need help?
- [ ] **Task board** - Anything stuck in one status too long?

## Actions
- If @mentioned or assigned: Respond/act immediately
- If blockers detected: Help unblock
- If nothing urgent: Report `HEARTBEAT_OK`
- If significant activity: Update WORKING.md

## Quiet Hours (23:00-08:00 NZDT)
- Only respond to urgent @mentions
- Otherwise: `HEARTBEAT_OK`
